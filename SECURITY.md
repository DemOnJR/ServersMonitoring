# Security Policy

## Supported Versions

Only the **latest released version** of Server Monitor is supported
with security updates.

Older versions may contain known vulnerabilities and are not supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Please report it responsibly by contacting:

**pipiasibogdan@gmail.com**

Include as much information as possible:
- A clear description of the issue
- Steps to reproduce
- Affected version
- Potential impact (if known)

You will receive an acknowledgment within **72 hours**.

---

## Disclosure Policy

- Valid security reports will be investigated as soon as possible
- Confirmed vulnerabilities will be fixed and released in a new version
- Credit will be given to the reporter (unless you prefer to stay anonymous)

---

## Scope

This security policy applies **only to the open-source core** of Server Monitor.

The hosted version is governed by separate Terms of Service and internal
security procedures.
